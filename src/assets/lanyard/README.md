# Lanyard Assets

Please download the following files and place them in this folder:

1. **card.glb** - The 3D model file for the card
   - You can edit this file at: https://modelviewer.dev/editor/
   - Download from the react-bits repository or create your own

2. **lanyard.png** - The texture image for the lanyard's band
   - Can be edited in any image editor
   - Download from the react-bits repository

These files are required for the Lanyard component to work properly.
